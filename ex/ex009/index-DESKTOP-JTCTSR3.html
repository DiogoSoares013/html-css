Listas ordenadas

Para criar listas ordenadas basta usar a tag ol e li para colocar os itens

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item

podemos usar parametros dentro da ol para listar de outras formas como foi feito acima

Tambem podemos fazer subitem dentro da lista

  1. Primeiro item
    1. subitem primeiro
    2. subitem segunto
  2. Ultimo item
